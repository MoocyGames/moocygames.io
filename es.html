Política de Privacidad

1. Introducción

En MOOCY GAMES, priorizamos la privacidad de nuestros usuarios. Esta política de privacidad explica cómo recopilamos, usamos y protegemos los datos personales durante el uso de nuestra aplicación.

2. Recopilación de Datos

Podemos recopilar los siguientes datos durante el uso del juego:

Información Personal:

Podemos recopilar información como nombre, dirección de correo electrónico y nombre de usuario para crear una cuenta y mejorar la experiencia del usuario.

Información del Dispositivo:

Podemos recopilar información técnica como el tipo de dispositivo, sistema operativo, dirección IP e identificadores únicos del dispositivo.

Datos del Juego:

Podemos registrar el rendimiento del usuario, logros y preferencias relacionadas con la experiencia de juego.

3. Propósito del Uso de Datos

Los datos que recopilamos se utilizarán para los siguientes propósitos:

Mejorar el rendimiento del juego y proporcionar una experiencia de juego personalizada.

Proporcionar soporte al cliente.

Desarrollar actualizaciones y nuevas características para el juego.

Cumplir con las obligaciones legales.

4. Compartir con Terceros

Podemos compartir datos del usuario con terceros solo en las siguientes circunstancias:

Con proveedores de servicios de análisis para mejorar el rendimiento del juego.

Con autoridades, si es requerido por la ley.

Con el consentimiento del usuario.

5. Período de Retención de Datos

Los datos del usuario se retendrán solo durante el tiempo necesario para sus fines previstos y se eliminarán de forma segura cuando ya no se necesiten.

6. Derechos del Usuario

Los usuarios tienen el derecho de acceder, corregir, eliminar o restringir el procesamiento de sus datos personales.

7. Seguridad

Tomamos todas las medidas técnicas y administrativas necesarias para proteger los datos del usuario. Sin embargo, tenga en cuenta que la transmisión de datos a través de Internet no es completamente segura.

8. Privacidad de los Niños

El juego no está destinado a niños menores de [13/16] años y no recopilamos intencionadamente información personal de niños. Si descubrimos que hemos recopilado datos personales de un niño, eliminaremos dicha información de inmediato.

9. Cambios en la Política

Esta política de privacidad puede actualizarse de vez en cuando. Cualquier cambio se comunicará a través de esta página.
